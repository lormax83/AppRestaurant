# AppRestaurant

[TP M1 MIAGE] CAVALLO - HERON


Application restaurant

Pour faire fonctionner le projet :

Importer avec git le dossier AppRestaurants

Pour le serveur : 

1/ Se placer dans le dossier serveur
2/ Lancer la commande 'node serveurCrudWithMongo.js' dans le terminal
Le node module est fourni directement dans le projet 


Pour le client :

1/ Se placer dans le dossier 'client'
2/ Faire 'npm install' pour générer le node-modul
3/ Faire 'npm run dev' pour lancé le client


Pour la partie BDD :

1/ Créer 3 collections 'main', 'dessert', 'starter'
2/ Importer respéctivement les  fichier JSON  du dossier 'MongoDB collection' dans leurs collections respectives


Le projet se découpent en 3 partie :

Choisissez un Restaurant puis cliqué sur Detail
En bas de la fenétre apparaitra les Details du Restaurant
Vous pourez alors Voir le Menu.
En bas de l'écran s'affichera le Menu du Restaurant
Vous pourez choisir une Quantité de plat et voir le Total de la Commande s'afficher
